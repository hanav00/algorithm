<h1>분수의 덧셈</h1>


<h2>문제 설명</h2>
정수 num1, num2가 매개변수로 주어질 때, num1를 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

<h2>제한 사항</h2>


- 0 < num1 ≤ 100
- 0 < num2 ≤ 100


<h2>입출력 예</h2>

|num1|num2|result|
|---|---|---|
|`3`|`2`|1|
|`10`|`5`|0|


<h2>입출력 예 설명</h2>


- 입출력 예 1) num1이 3, num2가 2이므로 3을 2로 나눈 나머지 1을 return 합니다.
- 입출력 예 2) num1이 10, num2가 5이므로 10을 5로 나눈 나머지 0을 return 합니다.
